# 🎮 MY GAME PORTAL

スマホのブラウザで遊べる、ゲームコントローラー専用のトレーニングゲーム集です。
Bluetoothで接続したコントローラー（PS5, Switch, Xboxなど）を使用して、反射神経やエイム力を測定・練習できます。

## 📱 遊び方
以下のURLにアクセスすると、ゲームポータル画面が表示されます。
**[ここをクリックしてゲームを遊ぶ](https://aifeiuehf-oss.github.io/reflex_mobile1/)**

※ 動作にはBluetoothコントローラーの接続が必要です。
※ 画面を開いた後、コントローラーのいずれかのボタンを押すと認識されます。

## 🕹️ 収録ゲーム

### 1. ⚡ REFLEX TEST (瞬発力測定)
ランダムに表示されるアイコンに合わせて、瞬時に正しいボタンを押すゲームです。
- **PSモード / Switchモード** 切り替え対応
- 視覚的な反射神経を鍛えます。

### 2. 🔘 RAPID EDGE OMEGA (連打・正確性)
指示されたボタンを正確に、かつ高速に連打するゲームです。
- **SINGLE**: 1つのボタンを連打
- **DUAL**: 2つのボタンを交互に連打
- **ORDER**: 光るボタンを順番に押す

### 3. 🎆 FIREWORKS SHOOTER (エイム練習)
スティック操作で動くターゲットを狙い撃つシューティングゲームです。
- スマホ画面対応の移動制限機能
- 花火のような爆発エフェクト付き
- 感度（Sensitivity）調整機能あり

## 🛠️ 使用技術
- HTML5 / CSS3
- JavaScript (Vanilla JS)
- Gamepad API (コントローラー入力の取得)
